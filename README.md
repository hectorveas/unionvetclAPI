Es un proyecto adaptado a la digitalizacion de hoy en dia para una clinica veterinaria que desea volver su entorno tecnologico mas amigable para ellos como para sus usuarios!

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
